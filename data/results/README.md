# Data Collection Results

Store your detection results here.

## Naming Convention
`YYYY-MM-DD_description_XXXch.json`

Example: `2025-11-11_week3_day1_100ch.json`

## Files
- `*_test_*.json` - Test runs (don't use for analysis)
- `*_week3_*.json` - Week 3 collection
- `*_combined_*.json` - Merged datasets
